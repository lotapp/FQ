<h1>①酸酸乳的windows客户端教程</h1>
<h3>1、安装 .NET Framework</h3>
<p>一般Windows 10 已经自带，不需要额外安装，如果您的系统版本略旧且无法打开软件，请点击<a href="https://www.microsoft.com/zh-tw/download/details.aspx?id=53345" target="_blank" rel="noopener">这里</a><br />适用于Windows 7 SP1、Windows 8.1、Windows Server 2008 R2 SP1、Windows Server 2012 及Windows Server 2012 R2 的Microsoft . NET Framework 4.6.2 (Web 安装程式)</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/830004357.png"><img class="alignnone size-medium wp-image-1251" src="https://cangshui.net/wp-content/uploads/2017/08/830004357-300x100.png" alt="" width="300" height="100" /></a></p>
<p><b>2、解压 酸酸乳 客户端</b></p>
<p>请打开下载的酸酸乳 压缩包文件解压，并双击运行酸酸乳-dotnet4.0</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/20170830004509.png"><img class="alignnone size-medium wp-image-1252" src="https://cangshui.net/wp-content/uploads/2017/08/20170830004509-300x114.png" alt="" width="300" height="114" /></a></p>
<p><b>3、配置 酸酸乳 账号</b></p>
<p>复制类似 “ssr://ODguMTk4LjU……………………” 这种格式的链接</p>
<p>在您右下方状态栏 酸酸 图标右键-服务器-从剪贴板导入ssr链接</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/001931.png"><img class="alignnone size-medium wp-image-1249" src="https://cangshui.net/wp-content/uploads/2017/08/001931-283x300.png" alt="" width="283" height="300" /></a></p>
<p>双击您右下方状态栏的酸酸乳 图标，即可看到您的节点信息和账号已经自动配置完成</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/5347.png"><img class="alignnone size-medium wp-image-1250" src="https://cangshui.net/wp-content/uploads/2017/08/5347-300x149.png" alt="" width="300" height="149" /></a></p>
<p>如果导入后没有反应仍旧是未代理状态、任务栏中的小飞机图标没有变成青色，按如图操作即可</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/2225.png"><img class="alignnone size-medium wp-image-1253" src="https://cangshui.net/wp-content/uploads/2017/08/2225-300x186.png" alt="" width="300" height="186" /></a></p>
<p>可根据自己的需求设置代理规则，绕过局域网即内网不使用代理，绕过大陆即国内网址不使用代理，我反正一般用全局，即所有网络链接都使用代理</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/2345.png"><img class="alignnone size-medium wp-image-1254" src="https://cangshui.net/wp-content/uploads/2017/08/2345-300x185.png" alt="" width="300" height="185" /></a></p>
<p>切换服务器，当你导入了多个酸酸乳服务器后可以切换使用，如果切换后未生效，可以右键点击小图标点退出，然后再启动即可</p>
<p><a href="https://cangshui.net/wp-content/uploads/2017/08/5002600.png"><img class="alignnone size-medium wp-image-1255" src="https://cangshui.net/wp-content/uploads/2017/08/5002600-300x219.png" alt="" width="300" height="219" /></a></p>
